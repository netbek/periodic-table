Periodic Table Data Files

All files are comma-separated values files.

Main Data File

File: pt-data1.csv

This file contains the following information on each row:

  • atomic number
  • symbol
  • name
  • atomic mass in au or g/mol
  • CPK color in RRGGBB hex format
  • electronic configuration
  • electronegativity in Pauling
  • atomic radius in pm
  • ion radius in pm
  • van der Waals radius in pm
  • IE-1 in kJ/mol
  • EA in kJ/mol
  • oxidation states
  • standard state
  • bonding type
  • melting point in K
  • boiling point in K
  • density in g/mL
  • metal or nonmetal?
  • year discovered

 

Main Data File without Oxidation States

File: pt-data2.csv

This file has all of the same data as the file above, except for the oxidation states. The reason for this is that the oxidation states field contains commas, so you will find it much easier to process this file since every comma necessarily means a new field.

Here is a file of oxidation states, with each row beginning with the atomic number followed by a comma-separated listing of the oxidation states.

 

Historic Information

File: pt-history.csv

This file contains the following on each line:

  • atomic number
  • symbol
  • name
  • year discovered
  • discoverer
  • history
